# Thin film interference: My Colab Analysis
IN this project, the python code will take in values of refractive indeces of the substrate and the film layer, the thickness of the film layer and the maximum and minimum wavelengths of light incident on the film-substrate setup and generates a graph of wavelength vs reflectivity to understand at specific thickness of the film, how much light is reflected at different wavelengths.
